# Sports Lottery Analysis (体彩数据分析)

基于 Vue 3 的体彩数据分析应用，可视化展示前区（蓝球）和后区（红球）的开奖频率趋势。

## 功能特性

- **数据可视化**：通过折线图直观展示前区蓝球（1-35）和后区红球（1-12）的出现频率。
- **频次分析**：详细统计每个号码在选定区间内的出现次数。
- **灵活筛选**：
  - 支持按期号区间筛选（起始期号 - 结束期号）。
  - 提供最近 20、50、100 期的快捷筛选按钮。
  - 默认展示最近 100 期数据。
- **列表排序**：支持按出现频次进行升序、降序排列，或恢复默认排序。
- **响应式界面**：使用 Element Plus 构建，包含吸顶表头和自适应图表布局。

## 技术栈

- **Vue 3**: 组合式 API (Composition API)
- **Vite**: 下一代前端构建工具
- **Element Plus**: 基于 Vue 3 的组件库
- **ECharts**: 强大的可视化图表库
- **Sass**: CSS 预处理器

## 快速开始

### 环境要求

- Node.js (推荐 v16+)

### 安装依赖

```bash
npm install
```

### 开发模式运行

```bash
npm run dev
```
